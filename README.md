# Farm Adventure Prototype

Прототип проекта Godot 4 с главным меню и первым уровнем в виде прокручиваемой фермы.

## Структура

- `project/project.godot` — конфигурация проекта Godot.
- `project/scenes/main_menu.tscn` — главное меню с кнопками "Начать игру" и "Выйти".
- `project/scenes/level_1.tscn` — сцена первого уровня с параллакс-фоном, тайловой землёй и точкой появления игрока.
- `project/scripts` — GDScript-скрипты для меню, уровня и фоновых слоёв.

## Запуск

1. Откройте папку `project` в Godot 4.2+.
2. Запустите проект — по умолчанию откроется главное меню. Нажмите «Начать игру», чтобы перейти на уровень.

## Кастомизация

- Для замены процедурных тайлов и фона замените соответствующие скрипты или подставьте собственные текстуры в `project/assets` и используйте их в сценах.
- Логику генерации тайлов можно изменить в `project/scripts/level_1.gd` (метод `_create_tiles_texture`).
